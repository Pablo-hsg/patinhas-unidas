# 🐾 Patinhas Unidas

Site institucional da ONG Patinhas Unidas, desenvolvido como projeto semestral da disciplina de Projeto em Desenvolvimento de Sistemas – Uninove 2026/1.

## 📋 Sobre o Projeto

Sistema WEB para a ONG fictícia **Patinhas Unidas**, dedicada ao resgate e adoção responsável de animais em São Paulo. O site oferece informações sobre a organização, animais disponíveis para adoção, formas de ajudar e canal de contato.

## 📄 Páginas do Sistema

| Página | Descrição |
|--------|-----------|
| `index.html` | Página inicial com hero, missão, animais em destaque e depoimentos |
| `pages/sobre.html` | História da ONG, valores e equipe |
| `pages/animais.html` | Listagem de animais com filtro por tipo e porte |
| `pages/como-ajudar.html` | Adoção, doação via PIX e voluntariado |
| `pages/contato.html` | Formulário de contato e informações |

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura semântica
- **CSS3** – Estilização com variáveis CSS e responsividade
- **JavaScript** – Interatividade (filtros, formulário, menu mobile)
- **Google Fonts** – Tipografia (Playfair Display + DM Sans)
- **Vercel** – Hospedagem na nuvem
- **GitHub** – Versionamento

## ✨ Funcionalidades

- ✅ Design responsivo (mobile, tablet, desktop)
- ✅ Navbar fixa com menu mobile
- ✅ Filtro de animais por tipo (cão/gato) e porte
- ✅ Formulário de contato com validação
- ✅ Botão de copiar chave PIX
- ✅ Animações ao scroll
- ✅ 5 páginas completas
